# Chiller Web Control

A Flask-based web dashboard for visualizing and interacting with the custom chiller system. It shows temperature readings, system status, and allows control of the setpoint and overrides.

## Features

- Web-based dashboard built with Flask
- Real-time temperature monitoring
- Controls for system override and setpoint adjustment
- Designed to run headless on a Raspberry Pi

## Usage

Start the Flask app:

```bash
python app.py
